# Nirvaan Cafe & Lounge

A beautiful single-page website for **Nirvaan Cafe & Lounge**, Vadodara.  
Built with HTML, TailwindCSS CDN, and GSAP scroll animations.

## 🚀 Deploy to Vercel

This is a **zero-build static site** — no install step needed.

1. Push this repository to GitHub
2. Go to [vercel.com](https://vercel.com) → **Add New Project**
3. Import the repository
4. Vercel will auto-detect the static config — click **Deploy**

> The site will be live at your Vercel URL instantly.

## ✏️ Before Going Live

- Update the phone number in `index.html` (search for `+919876543210`)
- The "Get Directions" link already points to the correct Google Maps address
